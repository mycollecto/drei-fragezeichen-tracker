# ❓️❓️❓️ Die drei Fragezeichen Tracker 🔍

Lesetracker für alle Die drei ??? Bücher und Hörspiele.

**Live:** https://mycollecto.github.io/drei-fragezeichen-tracker/

## Features
- 📚 **Über 300 Bände** in 4 Reihen
- 🖼️ Echte Cover vom Fandom-Wiki
- ✅ Abhaken mit Konfetti
- 🛒 **Amazon Affiliate** – Buch & Hörspiel Buttons pro Band
- 💾 Fortschritt lokal gespeichert
- 📱 PWA – auf Homescreen installierbar

## Reihenfolge

Die App folgt der **offiziellen Kosmos-Reihenfolge** – bestätigt durch den Kosmos-Verlag (2009, 2014) und dokumentiert auf Wikipedia sowie rocky-beach.com.

- Bände 1–55: US-Originale in Kosmos-Reihenfolge (weicht von US-Erscheinungsreihenfolge und Hörspiel-Zählung ab)
- Band 56+: Deutsche Fortsetzung (Kosmos-Nummerierung)
- Jubiläumsbände 100, 125, 150, 175, 200 = je 3 Teile unter einer Nummer
